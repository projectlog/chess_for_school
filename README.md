# 체스 첫걸음

10~14세 체스 초보자를 위한 교육용 웹 앱입니다.

## 실행 방법

1. 이 폴더의 모든 파일을 같은 위치에 둡니다.
2. `index.html`을 브라우저에서 엽니다.

## GitHub Pages 배포

1. 새 GitHub 저장소를 만듭니다.
2. `index.html`, `style.css`, `script.js`를 저장소 루트에 업로드합니다.
3. 저장소의 **Settings → Pages**에서 배포 브랜치를 `main`, 폴더를 `/root`로 선택합니다.
4. 생성된 GitHub Pages 주소로 접속합니다.

## 주요 기능

- 체스판과 기물 규칙 8단계 학습
- 기물별 이동 가능 칸 시각화
- 초보자용 미션 5개
- 친구와 2인 대국
- 간단한 컴퓨터 상대 대국
- 체크, 체크메이트, 스테일메이트 판정
- 캐슬링, 앙파상, 폰 승급 지원
- 모바일 반응형 화면
- 외부 라이브러리 없는 바닐라 HTML/CSS/JavaScript
